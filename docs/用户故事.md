# 用户故事（User Stories）

## Reporter（现场信息员）

- **R-1** 创建新事件
- **R-2** 文字上报
- **R-3** 拍照上报（一次最多9张）
- **R-4** 语音上报（自动转写文字）
- **R-5** 查看当前快报摘要
- **R-6** 离线可用，有网自动同步

## Reviewer（编辑审核员）

- **V-1** 接收AI初稿通知
- **V-2** 在线编辑修改
- **V-3** 变化对比（本版 vs 上版差异高亮）
- **V-4** 手动即时生成（不等10分钟周期）
- **V-5** 一键定稿
- **V-6** 导出Word/PDF
- **V-7** 按时间倒序查看所有历史版本

## Commander（指挥官）

- **C-1** 默认只看最新定稿版
- **C-2** 回溯历史定稿版本
- **C-3** 多事件快速切换

## Admin（系统管理员）

- **A-1** 成员管理（添加/删除/修改角色）
- **A-2** 事件管理（创建/关闭/归档）
- **A-3** 快报模板配置
