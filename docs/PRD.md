# PRD 核心框架：应急快报

## 1. 产品概述

面向灾害事故现场的多人协作信息快报系统。50人通过华为手机采集文本、语音、视频、照片等现场信息，AI后台每10分钟自动整合提炼出一版结构化快报，支持编辑审阅后一键报送上级单位。

## 2. 用户与角色

| 角色 | 人数 | 职责 | 终端 |
|------|------|------|------|
| 现场信息员 | ~45人 | 发文字、拍照、发语音、录视频 | 华为手机 App |
| 编辑审核员 | ~3人 | 审阅AI初稿、调整措辞、定稿 | 手机/PC网页 |
| 指挥官/决策层 | ~2人 | 只看最终定稿版本 | 手机/PC均可 |
| 系统管理员 | 1人 | 创建项目、配置模板、管理成员 | PC后台 |

## 3. 核心工作流

现场人员上报素材 → AI每10分钟整合生成 → 编辑审阅定稿 → 导出Word报送上级

## 4. MVP 分期

| 阶段 | 内容 |
|------|------|
| P0（必备） | 文字+照片上报 / 10分钟自动生成 / 编辑审阅定稿 / 导出Word |
| P1（重要） | 语音自动转写 / 视频处理 / 变化标注 |
| P2（增强） | 离线模式 / 矛盾检测 / PC审阅后台 / 多事件管理 |
